My Projects

Dice Game

I built a dice game where you click a button to roll two dice, and the screen updates with new images and the result. I used JavaScript, HTML, and CSS to create this game. From this project, I learned how to use JavaScript to update the page automatically without always refreshing it. During this process, I learned and tried various if-else statements to understand how web elements work and to finish this project.

Study Buddy

I built a weekly planner where students can organize their study schedule. I used Bootstrap to add buttons, rows, and columns, and learned how to make organized layouts. I also learned how to make cards and how they add to the appeal of the website. I saw how on different screens, websites can look different and ultimately learned how to create a responsive site that is appealing for users and neat.

Wedding Planner

I created a wedding page that shows event details and guest information. I used Bootstrap to organize sections, cards, and place images. This helped me learn how to organize and develop various sections of a site so that it is organized and is easy for user readability. This is essential for havign an organized site that benefits clients
